## [작업 계획]

### 주제 설정
**"고등학생의 역사적 사고력 향상을 위한 1차 사료 기반 캠페인 — 사전·사후 설문을 통한 효과 검증"**

근거:
- 학생의 진로(사학과)와 직결되는 "사료 해석 능력" "역사적 사고력" 주제
- 학생이 원하는 설문→개입→재설문 구조에 정확히 부합
- 1학년 때 숏폼/영상 형식이 역사 이해에 미치는 영향을 탐구한 이력과 자연스럽게 연결
- 기존의 "소음 연구" 봉사활동 방법론(설문→포스터→피드백)을 역사 분야로 심화 확장
- "사료를 직접 읽고 해석하는 경험"이라는 구체적 개입이 가능하여 캠페인 활동이 명확함

### 활동 설계 (보고서에 서술할 내용)
1. **사전 설문**: 같은 반/학년 학생 대상, 역사적 사고력 관련 문항 (사료 해석 경험, 역사 사건을 다각도로 보는 습관, 1차 사료와 2차 자료 구분 능력 등)
2. **문제 인식**: 사전 설문 결과 분석 → 대부분의 학생이 교과서 서술을 그대로 수용하며 1차 사료를 직접 읽어본 경험이 거의 없음을 확인
3. **개입 활동**:
   - 1차 사료(조선왕조실록 원문, 독립운동 관련 원사료 등) 발췌본을 활용한 "사료 읽기 워크숍" 또는 포스터 캠페인
   - 복수의 사료를 비교하며 같은 사건에 대한 서로 다른 기록/해석을 경험하게 하는 활동
4. **사후 설문**: 동일 문항으로 재조사 → 사료에 대한 이해도, 역사적 사고력 변화 측정
5. **결과 분석 및 제언**

### Word 문서 구성 (양식 항목 순서대로)
1. **주제**: 한 문장으로 명확하게
2. **중요 키워드**: 5~7개
3. **동기**: 본인의 역사학 탐구 경험, 이전 활동에서의 문제의식 연결
4. **탐구 내용**: 가장 핵심 — 사전 설문 설계/결과 → 캠페인 활동 내용 → 사후 설문 결과 → 비교 분석 (분량의 60% 이상)
5. **느낀점, 흥미로웠던 점, 더 탐구하고 싶은점**: 개인적 성찰, 사학도로서의 성장
6. **핵심만 요약**: 3~5문장으로 압축

- Word MCP 도구로 .docx 생성
- 폰트: Malgun Gothic, 11pt, 줄간격 1.5
- 본문 양끝 맞춤(justify)
- 파일명: result.docx
