# [작업 계획]

## 주제 방향
**희망 진로**: 반도체공학(자) → **진로에서 활용되는 AI**: 반도체 설계/제조 공정에서의 AI 기술

구체적으로, **AI 기반 반도체 설계 자동화(EDA AI)** 또는 **반도체 공정 결함 검출 AI** 중 택 1. 학생의 반도체·신소재 탐구 이력과 자연스럽게 연결되며, 기술 정의·원리·사례·긍정/부정 측면 모두 구체적으로 작성 가능.

## 작성 순서
1. **항목 1** — 희망 진로: 반도체공학자 (반도체 설계 및 공정 엔지니어)
2. **항목 2** — 진로에서 활용되는 AI: AI 기반 반도체 결함 검출 기술 (또는 AI-EDA)
3. **항목 3** — 표 형식 3행 이상:
   - 기술의 정의 (필수)
   - 동작 원리 (딥러닝 기반 이미지 분류/패턴 인식)
   - 실제 활용 사례 (삼성전자, TSMC 등 팹에서의 웨이퍼 검사)
   - 구성 요소 (CNN, 이미지 센서, 데이터 파이프라인)
4. **항목 4** — 긍정적 측면 3가지: 검출 정확도 향상, 공정 시간 단축, 수율 향상
5. **항목 5** — 부정적 측면 3가지: 학습 데이터 편향, 고비용 인프라, 일자리 대체
6. **항목 6** — 개선 방안 3가지: 데이터 다양성 확보, 단계적 도입, 인간-AI 협업 체계
7. **항목 7** — 탐구 소감: 느낀점 / 새롭게 알게된 점 / 더 알아보고 싶은 내용 (반도체 탐구 이력과 연결)
8. **항목 8** — 자료 출처 2~3건 (기사, 논문 등 실존 확인)
9. **AI 활용 기재란** — 비워둠 (요청사항 반영)

## 산출물 형식
- **result.md**: 양식 항목별 텍스트 원문 (학생이 HWP에 복사·붙여넣기 가능)
- **result.docx**: Word MCP로 동일 내용 .docx 파일 생성 (백업용)
- **result.json**: outputs 배열 포함
